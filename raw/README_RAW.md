# Raw Data

The full original raw archive is not included in the core package by default to keep the public release compact and easy to navigate. The analysis-ready dataset, channel reconstructions, audit tables, and result exports are included.

If releasing the original raw files, place the archive here and name it:

`rao_jiaojiao_raw_data_archive.zip`

The reconstruction rules are documented in `docs/DATA_DESCRIPTION.md` and audit tables under `data/audit/`.
