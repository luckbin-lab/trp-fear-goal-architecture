# Open Data Notes

## Recommended public release structure

For journal submission and public archiving, release this package as the main open-data/code repository. If desired, raw source files can be released as a separate archive because their directory names and encodings are heterogeneous and may be difficult for users to navigate.

## Raw data

The current open package includes analysis-ready data, channel-level reconstructions, audit tables, result tables, figures, and scripts. The original raw archive can be deposited separately as `raw_data_archive.zip` with a README explaining the source software:

- FreezeFrame exports for freezing.
- DeepSqueak/Matlab exports for USV calls.
- Graphic State Notation CSV exports for lever logs.
- Excel files for structured boli records.
